Dataset sources
Norman MacDonald norman@cs.dal.ca

The data files in this folder are compiled from the following sources:

Genotype data (genotype_prokaryote.cv1-2, genotype_prokaryote.cv2-2, genotype_prokaryote.profile)
STRING 8: (http://www.string-db.org/)

Phenotype data (phenotype.profile)
DOE JGI IMG (http://img.jgi.doe.gov/cgi-bin/pub/main.cgi)
Non-halophilic data from NCBI lproks: (http://www.ncbi.nlm.nih.gov/genomes/lproks.cgi)

Taxonomic data (taxonomic_groups.txt, taxonomic_groups_propagated.txt)
NCBI Taxonomy (http://www.ncbi.nlm.nih.gov/Taxonomy/)
